<objective>
In the job-leads intro-matches UI (RecommendationCard on the job lead drilldown page), make each contact name a clickable link to that contact's record page, and add an "Override" button next to the LinkedIn badge that excludes the contact from future intro recommendations by setting `doNotUseForIntros=true`.

Purpose: Lets the owner jump from a recommendation straight to the contact record, and prune contacts they don't want surfaced as intro paths — `buildRecommendations` already hard-excludes flagged contacts, so the override takes effect on the next load and is reflected immediately via optimistic removal.

Output: Updated RecommendationCard (name link + Override button) and RecommendationList (contactId threading + override handler).
</objective>

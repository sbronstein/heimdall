D-08 state machine (from CONTEXT.md):

  • pending → scraping (existing job-page-scrape flow)
  • scraping → scraped (existing) or scraping → pending (existing fallback on scrape error)
  • scraped → queued (new: POST /search becomes thin flip)
  • queued → searching (skill claims)
  • searching → found (skill success — writes prospects)
  • searching → failed (skill failure — writes lastError)
  • failed → queued (retry — clears lastError)
  • found → ready / actioned / archived (existing)
  • ready → actioned / archived (existing)
  • actioned → archived (existing)
  • archived: terminal

```typescript
async function sha256Hex(input: string): Promise<string> {
  const buf = new TextEncoder().encode(input);
  const digest = await crypto.subtle.digest('SHA-256', buf);
  return Array.from(new Uint8Array(digest))
    .map((b) => b.toString(16).padStart(2, '0'))
    .join('');
}
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Claude Code (skill) → /api/* | Untrusted-by-default header; bearer token from `~/.heimdall/api-token` crosses into the Next.js middleware |
| Filesystem → process | `~/.heimdall/api-token` (chmod 600) is the canonical token storage; readable only by the owning user |
| `.env.local` → process.env | `API_TOKEN_HASH` (hash, never plaintext) and `SINGLE_USER_EMAIL` reach the middleware via environment loading |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-02-01 | Spoofing | Bearer token bypass on /api/* | mitigate | SHA-256 hash compared via `crypto.subtle.digest` (timing-attack resistance is acceptable here — single-user deployment, no concurrent attackers); plaintext token never stored on disk in repo or env (only the hash) |
| T-05-02-02 | Spoofing | Multi-tenant deployment misuse | mitigate | Explicit `process.env.SINGLE_USER_EMAIL === ALLOWED_EMAIL` gate refuses the bypass on any deployment where SINGLE_USER_EMAIL is unset or differs (per D-21) |
| T-05-02-03 | Information Disclosure | Token in logs | mitigate | Middleware short-circuits silently on bypass success; the generation script does not print the plaintext token to stdout (only the hash and the file path); token file is chmod 600 (owner-read-only); .env.local is gitignored |
| T-05-02-04 | Elevation of Privilege | Token replay | accept | Single-user app on a Vercel deployment + local Chrome; threat model accepts the long-lived token as a per-machine secret. Rotation is manual (regenerate file + update env var). YAGNI for revocation/expiry per CONTEXT.md `<deferred>` |
| T-05-02-05 | Tampering | env.example.txt accidental real-secret commit | mitigate | env.example.txt contains placeholder text (`API_TOKEN_HASH=`) with `# Example:` comments, never real values; `.env.local` is gitignored (preserved from prior phases) |
</threat_model>

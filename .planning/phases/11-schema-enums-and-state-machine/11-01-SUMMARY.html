drizzle/schema/outreach-emails.ts (new) — outreach_emails table with:

  • FKs: campaign_id → outreach_campaigns.id, contact_id → contacts.id
  • Channel: outreach_channel enum default 'email', NOT NULL
  • Content: generated_subject/body + edited_subject/body (nullable; final = editedX ?? generatedX)
  • Status: outreach_email_status enum default 'pending', NOT NULL
  • Gmail linkage: gmail_draft_id (nullable, Phase 17)
  • Failure tracking: last_error, last_error_at (timezone, mirrors job-leads.ts:36-38)
  • Lifecycle timestamps: generated_at, approved_at, drafted_at (timezone, nullable)
  • UNIQUE (campaign_id, contact_id) constraint
  • 3 indexes: outreach_emails_campaign_id_idx, outreach_emails_status_idx, outreach_emails_contact_id_idx

drizzle/schema/index.ts — Appended barrel re-exports after ./job-leads:

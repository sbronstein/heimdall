- POST /api/outreach-campaigns  body `{ name: string, goalInstruction: string }`
  → `{ success: true, data: { id, name, goalInstruction, status, createdAt, updatedAt } }`

  → `{ success: true, data: { inserted: number, skipped: number } }`
  Dedup is server-side (onConflictDoNothing + unique (campaign_id, contact_id)) → criterion #5.
  ⚠ If selectedIds.size > 500, chunk into batches of <=500 sequential POSTs to the same endpoint.
